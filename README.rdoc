== README

Following steps are necessary to get the script up and running.

* Open terminal and run the following commands

* <tt>git clone https://github.com/rokibulhassan/TaskManager.git</tt>
* <tt>cd TaskManager</tt>
* <tt>bundle install</tt>
* <tt>rake db:create</tt>
* <tt>rake db:migrate</tt>
* <tt>rails s</tt>

* Please do registration and login, To see the demo browse this url https://thawing-wildwood-54815.herokuapp.com
